# Data

This directory is **not committed to git**. Raw and processed data must be downloaded locally.

## Raw data (`data/raw/`)

### UCI Bone Marrow Transplant Dataset

- **Source**: https://archive.ics.uci.edu/dataset/565/bone+marrow+transplant+children
- **File**: `bone-marrow.csv`
- **Description**: 187 pediatric patients who underwent allogeneic hematopoietic stem cell transplantation. Contains donor/recipient HLA typing (antigen and allele level), clinical covariates, and outcomes (survival time, GvHD, relapse, cause of death).
- **Download**: Manual download from UCI ML Repository (requires registration).

### IPD-IMGT/HLA Protein Sequences

- **Source**: https://www.ebi.ac.uk/ipd/imgt/hla/
- **File**: `hla_sequences.json`
- **Description**: Mapping from HLA allele name → amino acid sequence for all known HLA alleles.
- **Download**: Run `uv run python scripts/download_hla_seqs.py` (implementation pending).

## Processed data (`data/processed/`)

Generated by `scripts/preprocess.py`:

| File | Description |
|---|---|
| `train.parquet` | Training split |
| `val.parquet` | Validation split |
| `test.parquet` | Test split (held out until final evaluation) |
| `embeddings.h5` | HDF5 cache of ESM-2 HLA allele embeddings |

## Data provenance

All downstream results must be reproducible from the raw data files above given a fixed random seed. The random seed is set in `capa/config.py` (default: 42) and overridable via the `CAPA_DATA_RANDOM_SEED` environment variable.
